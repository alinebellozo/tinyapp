## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session
- nodemon
- mocha (tests)
- chai (tests)

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `run npm` command.
